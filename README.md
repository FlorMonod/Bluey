# Sitio Web Estático – Primer Parcial

**Primer parcial individual**  
Trabajo práctico realizado en el marco de la **Tecnicatura Universitaria en Programación – UTN**.

---

## Descripción
Desarrollo de un **sitio web estático de baja complejidad**, realizado como primer parcial de la materia, utilizando tecnologías web básicas.

El sitio está orientado a la presentación de una **serie / película / novela**, cumpliendo con los requisitos funcionales solicitados por la cátedra.
Link: https://flormonod.github.io/Bluey/
---

## Tecnologías utilizadas
- HTML  
- CSS  
- JavaScript  

---

## Estructura del sitio
El sitio cuenta con la siguiente estructura:

### Página de inicio
- Archivo principal: `index.html`
- Presentación general del tema elegido

### Galería de imágenes
- Página funcional
- Implementación de estilos con CSS
- Funcionalidad mediante JavaScript

### Página de contacto
- Formulario funcional
- Integración de mapa de Google

### Página distribuidora
- Página de primer nivel que redirige a un segundo nivel
- Al menos **3 subpáginas**, por ejemplo:
  - Temporadas
  - Capítulos
  - Secciones relacionadas al contenido principal

---

## Publicación
El proyecto se encuentra:
- Publicado mediante **GitHub Pages**, permitiendo su navegación online además del acceso al código fuente

---

## Objetivo académico
- Aplicar conceptos básicos de desarrollo web
- Estructurar un sitio con múltiples niveles de navegación
- Integrar HTML, CSS y JavaScript
- Publicar un proyecto web utilizando GitHub Pages
